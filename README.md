NEKO+PHP build pack

STATUS: WORK-IN-PROGRESS
------------------------

This is a build pack bundling NEKO and Apache for Heroku apps.

Configuration
-------------

The config files are bundled with the build pack itself:

* conf/httpd.conf

web root : web/

neko projects are detected by putting a index.n file in the web directory ( web/index.n )
